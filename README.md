ListRenderer Tutorial
=====================

Sources du tutoriel ListRenderer disponible sur http://www.tuto-codenameone.ch


Informations
=====================

Les librairies ne sont pas incluses dans le projet (elles prennent beaucoup d'espace disque).
Liste des librairies à ajouter :
- JavaSE.jar
- lib/CLDC11.jar
- lib/CodenameOne.jar
- lib/CodenameOne_SRC.zip

Créez un projet Codename One depuis votre IDE pour les obtenir facilement.
